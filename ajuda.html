Ternario

o ternario e composto por: {condição} ? {retorno se a consição for verdadeira} : {retorno se a consição for false}

Exermplo: let nome = pessoa.nome != null ? pessoa.nome : 'Sem Nome';

Em um ternario as condições são SEMPRE antes do ?.

Se tiver que testar a mesma variavel mais de uma vez, tem que repetir a parte da variavel.

Exemplo: let podeBeber = pessoa.idade >= 18 && pessoa.idade <= 60 ? true : false;

Veja que no exemplo acima, a pessoa so pode beber se tive a idade maior ou igual que 18 e menor ou igual a 60; Então em cada comparação tenho e referenciar "pessoa.idade" , não existe comparação [pessoa.idade>=18 && <=60] .

If e else

uma comparação de if e else e feita para execultar trecho de codigo caso uma condição seja verdadeida ou falsa. Veja que para existir um if não e necessario ter um else, porém para exisitr um else e obrigatorio existir um if.

Exemplo if sem else:
        if(pessoa.nome != null){
           let nome = pessoa.nome
        }
Exemplo if e else:
        if(pessoa.nome != null){
           let nome = pessoa.nome
        }else
           let nome = 'Sem Nome'
        }

Veja que em um if/else a condição SEMPRE fica dentro () do if, e o else NUNCA tera ()

Comparação

existe 2 tipos de verificador: && / ||

&& usamos quando um ternario ou um if(), tem mais de uma condição para ser testada e todas as condições devem ser verdadeiras.

Exemplo: let nomeTeste = pessoa.nome != null && pessoa.idade == 10 ? pessoa.nome : 'Sem Nome';

No caso acima a variavel nomeTeste,so vai receber o valor de pessoa.nome, se pessoa.nome não for null e a idade for igual a 10, as duas condições tem que ser verdadeira

|| usamos quando um ternario ou um if(), tem mais de uma condição para ser testada e apenas uma das condições devem ser verdadeiras .

Exemplo: let nomeTeste = pessoa.nome != null && pessoa.idade == 10 ? pessoa.nome : 'Sem Nome';

No caso acima a variavel nomeTeste, vai receber o valor de pessoa.nome, se pessoa.nome não for null ou idade for igual a 10, apenas uma da duas condições tem que ser verdadeira

para fazer comparações temos os operadores == / != / > / < />= / <=

'==' esse operador usamos para compara se um valor e IGUAL ao outro. Exemplo: 'Tiago' == 'Tiago' - resultado true / 'Tiago' == 'Jose' - resultado false

'!=' esse operador usamos para compara se um valor e DIFERENTE ao outro. Exemplo: 'Tiago' != 'Tiago' - resultado false / 'Tiago' != 'Jose' - resultado true

'<' esse operador usamos para compara se um valor e MENOR ao outro. Exemplo: 10 < 20 - resultado true / 10 < 10 - resultado false

'<=' esse operador usamos para compara se um valor e MENOR OU IGUAL ao outro. Exemplo: 10 <= 8 - resultado false / 10 <=10 - resultado true

'>' esse operador usamos para compara se um valor e MAIOR ao outro. Exemplo: 10 > 5 - resultado true / 10 > 10 - resultado false

'>=' esse operador usamos para compara se um valor e MAIOR OU IGUAL ao outro. Exemplo: 10 >= 12 - resultado false / 10 >= 10 - resultado true

Receber/Atribuir

quando falamos que uma variavel vai receber um valor, ou vair ser atribuidor um valor estamos falando do operador =

Vejamos se eu digo que a variavel nome receber uma string com valor Tiago estou dizendo que:

Exemplo: let nome = 'Tiago'

ou se eu digo nome recebe o valor Jose, represento:

Exemplo: let nome = 'Jose'

ou se eu digo atribui a variavel nome o valor Alysson, quero dizer:

Exemplo: let nome = 'Alysson'

Array []

Um array e simbolizado por [], o array e como uma lista, onde vc pode inserir varios objetos do mesmo tipo.

um array sempre inicia em 0
Podemos criar um array da seguinte forma let arrayNome: string[] = []; nesse exemplo criamos um array de string vazio

Exemplo criando array: let arrayNome: string[] = [];
Exemplo adicionando itens no array:
arrayNome.push('Tiago');
arrayNome.push('Jose');
arrayNome.push('Alysson');

Veja que acima criamos um array de string vazio, logo em seguida inserimos 3 objetos no array. Lembrando que o array sempre inicia em 0, vamos ter que vai ate a posição 2

pensando em um cenario onde eu tenha que setar o valor true em uma variavel apenas se na posição 1 tenha o valor 'Alysson', vamos recuperar o valor dessa posição da seguinte maneira:

Exemplo recuperar o valor de uma posição especifica= arrayNome[1]

então para criar uma condição vendo um objeto da lista, fazemos uma comparação da seguinte forma:

variavelBooleana = arrayNome[1] == 'Alysson' ? true : false;

No exemplo acima iremos atribuir a variavelBooleana o valor false, pois na posição [1] do array temos Jose.

porém pode surgir uma outra necessidade, de verificar se na posição 0 tem Tiago e na 2 tem Jose, podemos comparar da seguinte forma:

variavelBooleana = arrayNome[0] == 'Tiago' && arrayNome[2] == 'Jose' ? true : false;

No exemplo acima iremos atribuir a variavelBooleana o valor true, pois as duas condições são verdadeiras.
